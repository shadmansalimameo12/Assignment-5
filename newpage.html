Question-1: What are the different ways to select an element in the DOM?

ID : document.getElementById("id"), Class : document.getElementsByClassName("class"), Tag Name : document.getElementsByTagName("tag"), Query Selector: document.querySelector("css-selector"), Query Selector All : document.querySelectorAll("css-selector"), Name Attribute : document.getElementsByName("name")

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Gets or sets the content including HTML, innerText: Gets or sets only visible text (ignores hidden elements), textContent: Gets or sets all text (including hidden) but ignores HTML.

Question-4: What is event bubbling in the DOM?

Event Bubbling: When an event on a child element first triggers its own handler and then propagates up to its parent and ancestors.

Question-5: How do you create, add, and remove elements using JavaScript?

Create Element: document.createElement("tag"), Add Element: parent.appendChild(child) or parent.append(child), Remove Element: element.remove() or parent.removeChild(child)
